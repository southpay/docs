## Payment lifecycle

A payment moves through a predictable set of states from creation to confirmation.

| Status | Description |
|--------|-------------|
| `pending` | Created, waiting for customer action |
| `processing` | Funds received on-chain, confirming |
| `paid` | Confirmed and credited |
| `failed` | Could not be completed |
| `expired` | Window closed before payment arrived |
| `refunded` | Funds returned |
