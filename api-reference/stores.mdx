<Warning>
  All endpoints require authentication. See [Authentication](/authentication) for details.
</Warning>

Base URL: `https://api.southpay.io/api/v2`

---

## List stores

<CodeGroup>
```bash Request
GET /stores
Authorization: Bearer <API_KEY>
```

```json Response
[
  { "id": "store_abc", "name": "My Store" }
]
```
</CodeGroup>

---

## Create a store

<CodeGroup>
```bash Request
POST /stores
Authorization: Bearer <API_KEY>
Content-Type: application/json

{ "store": { "name": "My Store" } }
```

```json Response 201
{ "id": "store_abc", "name": "My Store" }
```
</CodeGroup>

---

## Update a store

<CodeGroup>
```bash Request
PATCH /stores/:id
Authorization: Bearer <API_KEY>
Content-Type: application/json

{ "store": { "name": "New Name" } }
```

```json Response
{ "id": "store_abc", "name": "New Name" }
```
</CodeGroup>

---

## Delete a store

```bash Request
DELETE /stores/:id
Authorization: Bearer <API_KEY>
```

Returns `204 No Content`.

---

## Switch active store

Switches the authenticated session context to a different store.

<CodeGroup>
```bash Request
POST /stores/:id/switch
Authorization: Bearer <API_KEY>
```

```json Response
{
  "message": "Switched to store",
  "store": { "id": "store_abc", "name": "My Store" }
}
```
</CodeGroup>

## Get store settings

<CodeGroup>
```bash Request
GET /stores/settings
Authorization: Bearer <API_KEY>
```

```json Response
{
  "enabled_asset_ids": ["ETH_USDC", "BTC_BTC"],
  "settlement_currency": "USD"
}
```
</CodeGroup>

---

## Update store settings

<CodeGroup>
```bash Request
PATCH /stores/settings
Authorization: Bearer <API_KEY>
Content-Type: application/json

{
  "store_settings": {
    "enabled_asset_ids": ["ETH_USDC", "BTC_BTC"],
    "settlement_currency": "EUR"
  }
}
```

```json Response
{
  "enabled_asset_ids": ["ETH_USDC", "BTC_BTC"],
  "settlement_currency": "EUR"
}
```
</CodeGroup>

**Request fields**

| Field | Type | Description |
| --- | --- | --- |
| `store_settings[enabled_asset_ids][]` | array | Asset IDs to accept. See [Supported Assets](/api-reference/assets) for valid values. |
| `store_settings[settlement_currency]` | string | ISO 4217 fiat code for settlement (e.g. `"USD"`) |
