## Get balances

Returns the current balance for each asset held by your store.

<CodeGroup>
```bash Request
GET /balances
Authorization: Bearer <API_KEY>
```

```json Response
[
  {
    "asset_id": "ETH_USDC",
    "coin_symbol": "USDC",
    "chain_symbol": "ETH",
    "atomic_decimals": 6,
    "balance_atomic": "1000000"
  },
  {
    "asset_id": "BTC_BTC",
    "coin_symbol": "BTC",
    "chain_symbol": "BTC",
    "atomic_decimals": 8,
    "balance_atomic": "50000"
  }
]
```
</CodeGroup>

**Response fields**

| Field | Type | Description |
| --- | --- | --- |
| `asset_id` | string | Unique identifier for the asset (use in API requests) |
| `coin_symbol` | string | Ticker of the coin (e.g. `USDC`, `BTC`) |
| `chain_symbol` | string | Blockchain network (e.g. `ETH`, `BTC`) |
| `atomic_decimals` | integer | Number of decimal places in the atomic unit |
| `balance_atomic` | string | Balance in atomic units (divide by `10^atomic_decimals` for display) |

## List supported assets

Returns all crypto assets available for accepting payments.

<CodeGroup>
```bash Request
GET /supported_assets
Authorization: Bearer <API_KEY>
```

```json Response
[
  {
    "asset_id": "ETH_USDC",
    "coin_symbol": "USDC",
    "chain_symbol": "ETH",
    "atomic_decimals": 6,
    "display_decimals": 2
  },
  {
    "asset_id": "BTC_BTC",
    "coin_symbol": "BTC",
    "chain_symbol": "BTC",
    "atomic_decimals": 8,
    "display_decimals": 8
  }
]
```
</CodeGroup>

Use `asset_id` values when configuring [store settings](/api-reference/stores#update-store-settings) or [payout details](/api-reference/payout-details).
