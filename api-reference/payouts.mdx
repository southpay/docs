<Warning>
  All endpoints require authentication. See [Authentication](/authentication) for details.
</Warning>

Base URL: `https://api.southpay.io/api/v2`

---

## List payouts

<CodeGroup>
```bash Request
GET /payouts?status=completed&asset_id=ETH_USDC
Authorization: Bearer <API_KEY>
```

```json Response
[
  {
    "id": "payout_abc123",
    "status": "completed",
    "amount_cents": 5000000,
    "asset": {
      "asset_id": "ETH_USDC",
      "coin_symbol": "USDC",
      "chain_symbol": "ETH"
    },
    "created_at": "2026-04-10T00:00:00.000Z",
    "updated_at": "2026-04-10T00:05:00.000Z"
  }
]
```
</CodeGroup>

**Query parameters**

| Parameter | Type | Description |
| --- | --- | --- |
| `status` | string | Filter by payout status |
| `asset_id` | string | Filter by asset (e.g. `ETH_USDC`) |

---

## Get a payout

<CodeGroup>
```bash Request
GET /payouts/:id
Authorization: Bearer <API_KEY>
```
